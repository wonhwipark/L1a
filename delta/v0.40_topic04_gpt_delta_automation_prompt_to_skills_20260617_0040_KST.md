## 2. Rationale

현재 5.x 파일들은 단순 프롬프트 문서가 아니라, Roo Code / Claude Code / 향후 Python 자동화 도구가 재사용할 수 있는 실행 스킬 성격이 강하다.

따라서 `automation_prompt`보다 `prompt`가 다음 방향과 더 잘 맞는다.

```text
- 공통 실행 원칙
- 환경 조사
- capability 조사
- 개별 자동화 수행 규칙
- 향후 Python CLI / npm wrapper로 발전 가능한 실행 단위
```
