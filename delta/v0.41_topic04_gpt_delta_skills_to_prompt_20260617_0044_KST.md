---

## 2. Rationale

현재 5.x 파일들은 실제 Claude/Roo runtime skill이라기보다, Python 자동화 도구 구현을 위한 프롬프트 및 명세 문서에 가깝다.

따라서 `skills/`는 `.claude/skills`, `.roo/skills` 같은 실제 실행 skill 폴더와 혼동될 수 있다.

현재 단계의 목적은 다음과 같다.

```text
- 자동화 프롬프트 정리
- Python 자동화 도구 구현 전 명세화
- Claude Code / Roo Code에 전달할 구현 지시서 관리
```

따라서 폴더명은 `prompt/`가 더 적절하다.
