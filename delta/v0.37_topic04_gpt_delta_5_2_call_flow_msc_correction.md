# v0.37 topic04 추가 GPT Delta — 5.2 Call Flow 출력 기준 수정

---

## 변경 사항

`prompt/5_2_code_analyzer_py_package_prompt.md`에서 Call Flow 산출물을 JSON이 아니라 MSC(PlantUML) 기준으로 수정하였다.

---

## 변경 이유

5.2 Code Analyzer의 목적은 HLD가 없는 기존 코드를 분석하여 문서화하는 것이다. 이때 Call Flow는 단순 machine-readable JSON보다 HLD에 바로 포함 가능한 MSC(PlantUML) 형태가 더 적합하다.

---

## 반영 내용

```text
Before:
- Call Flow JSON

After:
- Call Flow MSC(PlantUML)
```

구현 지시 프롬프트에도 다음 내용을 반영하였다.

```text
- models.py는 Call Flow MSC(PlantUML)를 출력 데이터 구조에 포함한다.
- call_flow_extractor.py는 추출된 구조에서 Call Flow MSC(PlantUML)를 생성한다.
- hld_generator.py는 Call Flow MSC(PlantUML)를 기반으로 HLD 업데이트 항목과 HLD 초안을 생성한다.
```
