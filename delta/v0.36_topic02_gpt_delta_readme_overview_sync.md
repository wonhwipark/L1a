## 2. 변경 목적

현재 패키지 기준으로 5.2 항목은 `Code Analyzer`로 정리되어 있으나, 일부 workflow 문서에는 과거 파일명인 `5_2_api_call_flow_db_py_package_prompt.md`가 남아 있다.

또한 `automation_specs/00_overview.md`의 기준 Master가 `v0.35`로 남아 있어 현재 Master인 `v0.36`과 불일치한다.

이번 Delta의 목적은 다음과 같다.

```text
1. readme_workflow.md의 automation_specs 파일 목록을 실제 파일명과 일치시킨다.
2. automation_specs/00_overview.md의 기준 Master 버전을 v0.36으로 동기화한다.
3. workflow에 S2A/S2B를 추가하여 GPT Delta 생성과 실제 패키지 업데이트를 분리한다.
4. workflow에 S2까지가 GPT 담당 영역임을 명시한다.
```
