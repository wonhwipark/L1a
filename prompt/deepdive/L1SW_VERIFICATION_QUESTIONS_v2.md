| 답변 결과 | 5.5 설계에 미치는 영향 |
|---|---|
| B-3: L1SW에 issue_type 매핑이 없음 | → 5.5의 `keywords.yaml`이 issue_type↔키워드 매핑의 유일한 SSOT가 됨 |

| C-1/C-2: fragment JSON 업데이트가 비공식적 | → 5.5의 누적 루프가 키워드 관리의 공식 채널이 됨 |
